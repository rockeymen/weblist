## 修改计划

### 1. 导航栏 Tab 居左对齐

**修改文件**: `styles.css`
**修改内容**:
- 将 `.nav` 类的 `justify-content` 从 `center` 改为 `flex-start`
- 保持其他导航栏样式不变

### 2. 排行榜布局调整

**修改文件**: `styles.css`
**修改内容**:
- 调整 `.leaderboard-content` 的布局，优化空间分配
- 调整 `.leaderboard-info` 的 flex 属性，给予更多空间
- 调整 `.leaderboard-likes` 的样式，使其更靠右
- 减小 `.rank` 与内容的间距

### 具体修改点

1. **导航栏样式**:
   ```css
   .nav {
       display: flex;
       gap: 1rem;
       flex-wrap: wrap;
       justify-content: flex-start; /* 改为居左对齐 */
       align-items: center;
       padding: 0.8rem 0;
       background-color: white;
       border-bottom: 1px solid #eee;
       box-shadow: 0 2px 5px rgba(0, 0, 0, 0.05);
   }
   ```

2. **排行榜内容布局**:
   ```css
   .leaderboard-content {
       display: flex;
       align-items: center;
       flex: 1;
       gap: 0.8rem;
       overflow: hidden;
   }
   
   .leaderboard-info {
       flex: 2; /* 增加中间文字空间 */
       min-width: 0;
   }
   
   .leaderboard-likes {
       flex-shrink: 0;
       margin-left: auto; /* 靠右对齐 */
   }
   
   .rank {
       font-size: 1.2rem;
       font-weight: 700;
       margin-right: 0.6rem; /* 减小左侧间距 */
       width: 25px;
       text-align: center;
       flex-shrink: 0;
   }
   ```

### 预期效果

1. 导航栏的 Tab 项将从居中对齐改为居左对齐
2. 排行榜项目中，中间的文字部分将获得更多空间
3. 点赞数部分将更靠右显示
4. 排名与内容的间距将减小
5. 整体布局更加紧凑，空间利用更合理

### 技术要点

- 使用 Flexbox 布局实现灵活的空间分配
- 通过 `margin-left: auto` 实现元素靠右对齐
- 调整 `flex` 属性值优化空间分配
- 保持响应式设计，确保在不同屏幕尺寸下都有良好显示效果
